# 更新日志

本文档记录了 Next Admin Starter 的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-01

### 新增

- 🎉 初始版本发布
- ✨ 基于 Next.js 15 + Ant Design 6 的完整后台模板
- 🎨 三种主题色切换（蓝猫、紫薯、橙子）
- 🎯 完整的布局系统（侧边栏、标签页、页面过渡）
- 🔒 基于路由的权限守卫系统
- 📦 核心业务组件（ProTable、SearchForm、ModalForm、DrawerForm）
- 🔌 完整的前后端分离架构
- 📝 详细的后端对接文档

### 功能

- 用户管理示例页面
- 数据分析页面
- 图表展示组件
- 表单处理组件
- 文件上传组件
- 图片预览组件
- 命令面板功能
- AI 助手功能
- 快捷键管理
- 热力图功能
- 性能监控
- 历史记录管理

### 技术栈

- Next.js 15 (App Router + RSC)
- React 19.2.4
- Ant Design 6.3.1
- TypeScript 5.8.3
- Tailwind CSS 4.2.1
- Zustand 5.0.3
- Axios 1.13.6

### 文档

- 完整的 README 文档
- AGENTS.md 项目上下文指南
- API_INTEGRATION.md 后端对接指南
- 详细的代码注释和类型定义

- **变更 (Changed)**: 功能变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug 修复
- **安全 (Security)**: 安全相关更新
